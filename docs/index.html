Installation

Install the SDK via Composer:

composer require sirv/sirv-rest-api-php

Requirements

  • PHP 7.4 or higher
  • ext-json
  • ext-curl
  • Guzzle HTTP client 7.0+

Quick Start

<?php

require_once 'vendor/autoload.php';

use Sirv\SirvClient;

// Initialize the client
$client = new SirvClient(
    'your-client-id',
    'your-client-secret'
);

// Get account information
$account = $client->account()->get();
echo "Account: " . $account['alias'] . "\n";

// List files in a directory
$files = $client->files()->list('/');
foreach ($files['contents'] as $file) {
    echo $file['filename'] . "\n";
}

Authentication

The SDK handles authentication automatically. When you make your first API call, it will obtain an access token using your client credentials. The token is automatically refreshed when it expires.

Note: Get your API credentials from the Sirv Dashboard. 
$client = new SirvClient(
    'your-client-id',
    'your-client-secret'
);

// Optional: Manually authenticate to verify credentials
try {
    $token = $client->authenticate();
    echo "Authenticated successfully\n";
} catch (\Sirv\Exception\AuthenticationException $e) {
    echo "Authentication failed: " . $e->getMessage() . "\n";
}

// Optional: Request a token with custom expiry time (5-604800 seconds)
$token = $client->authenticate(3600); // Token valid for 1 hour

Files API

Complete file management including upload, download, copy, rename, delete, and metadata operations.

File Operations

  • list() - List folder contents
  • getInfo() - Get file information
  • upload() - Upload a file
  • download() - Download a file
  • copy() - Copy a file
  • rename() - Rename/move a file
  • delete() - Delete a file
  • mkdir() - Create a folder
  • fetch() - Fetch from URL

Metadata Operations

  • getMeta() / setMeta()
  • getApproval() / setApproval()
  • getTitle() / setTitle()
  • getDescription() / setDescription()
  • getProductMeta() / setProductMeta()
  • getTags() / setTags() / deleteTags()

Batch Operations

  • zip() - Create ZIP file
  • getZipResult() - Get ZIP status
  • deleteBatch() - Delete multiple files
  • getDeleteBatchResult() - Get delete status

Media & Export

  • spinToVideo() - Convert spin to video
  • videoToSpin() - Convert video to spin
  • exportToAmazon()
  • exportToWalmart()
  • exportToHomeDepot()
  • exportToLowes()
  • exportToGrainger()

Basic Operations

// List folder contents
$files = $client->files()->list('/images');

// Upload a file
$result = $client->files()->upload('/local/path/photo.jpg', '/images/photo.jpg');

// Download a file
$content = $client->files()->download('/images/photo.jpg');
file_put_contents('downloaded.jpg', $content);

// Copy a file
$client->files()->copy('/images/photo.jpg', '/backup/photo.jpg');

// Delete a file
$client->files()->delete('/images/photo.jpg');

Search Files

// Search files
$results = $client->files()->search([
    'query' => 'filename:*.jpg',
    'from' => '/images',
    'size' => 100
]);

// Paginate through results
if (isset($results['_scroll_id'])) {
    $nextPage = $client->files()->searchScroll($results['_scroll_id']);
}

Metadata Operations

// Set file metadata
$client->files()->setMeta('/images/photo.jpg', [
    'title' => 'My Photo',
    'description' => 'A beautiful sunset'
]);

// Set tags
$client->files()->setTags('/images/photo.jpg', ['nature', 'sunset', 'beach']);

// Set product metadata
$client->files()->setProductMeta('/images/product.jpg', [
    'id' => 'SKU123',
    'name' => 'Product Name',
    'brand' => 'Brand Name'
]);

Batch Operations

// Create ZIP file
$job = $client->files()->zip([
    '/images/photo1.jpg',
    '/images/photo2.jpg'
], 'my-archive.zip');

// Check ZIP status
$result = $client->files()->getZipResult($job['token']);
if ($result['status'] === 'completed') {
    echo "ZIP URL: " . $result['url'] . "\n";
}

Account API

Manage your Sirv account settings and retrieve account information.

Available Methods

Method Description
get() Get account information including CDN URL and account name
update(array $settings) Update account settings (minify, fetching)
getLimits() Get API request limits and usage
getStorage() Get storage usage information
getUsers() List all account users
getBillingPlan() Get billing plan details
searchEvents(array $params) Search account events
markEventsSeen(array $eventIds) Mark events as seen

Examples

// Get account information
$account = $client->account()->get();

// Get API limits
$limits = $client->account()->getLimits();
echo "API calls used: " . $limits['used'] . "/" . $limits['limit'] . "\n";

// Get storage information
$storage = $client->account()->getStorage();
echo "Storage used: " . $storage['used'] . " bytes\n";

// Search account events
$events = $client->account()->searchEvents([
    'module' => 'files',
    'level' => 'error'
]);

Statistics API

Retrieve usage statistics for your account.

Method Description
getHttp($from, $to) Get HTTP transfer statistics
getSpinViews($from, $to, $alias) Get spin viewer statistics (max 5 days)
getStorage($from, $to) Get storage statistics 
// HTTP transfer statistics
$httpStats = $client->stats()->getHttp('2024-01-01', '2024-01-31');

// Using DateTime objects
$from = new DateTime('2024-01-01');
$to = new DateTime('2024-01-31');
$httpStats = $client->stats()->getHttp($from, $to);

// Storage statistics
$storageStats = $client->stats()->getStorage('2024-01-01', '2024-01-31');

User API

Retrieve user information.

// Get current user info
$user = $client->user()->get();

// Get specific user info
$user = $client->user()->get('user-id-here');

Error Handling

The SDK provides specific exception types for different error scenarios.

Exception Description
AuthenticationException Invalid credentials or authentication failed
ApiException General API errors
RateLimitException Rate limit exceeded
ValidationException Invalid request parameters 
use Sirv\SirvClient;
use Sirv\Exception\AuthenticationException;
use Sirv\Exception\ApiException;
use Sirv\Exception\RateLimitException;

try {
    $client = new SirvClient('client-id', 'client-secret');
    $files = $client->files()->list('/');
} catch (AuthenticationException $e) {
    echo "Auth error: " . $e->getMessage() . "\n";
} catch (RateLimitException $e) {
    echo "Rate limited. Retry after: " . $e->getRetryAfter() . " seconds\n";
} catch (ApiException $e) {
    echo "API error: " . $e->getMessage() . "\n";
    echo "HTTP Status: " . $e->getHttpStatusCode() . "\n";
}

RateLimitException Methods

  • getRetryAfter() - Seconds to wait before retrying
  • getRateLimit() - Rate limit for the endpoint
  • getRateLimitRemaining() - Remaining requests in window

ApiException Methods

  • getHttpStatusCode() - HTTP status code
  • getRequestId() - Request ID for debugging
  • getErrorDetails() - Additional error details

Configuration

Timeout

// Set custom timeout (in seconds)
$client = new SirvClient('client-id', 'client-secret', 60);

Custom Guzzle Options

// Pass additional Guzzle HTTP client options
$client = new SirvClient('client-id', 'client-secret', 30, [
    'proxy' => 'http://proxy.example.com:8080',
    'verify' => false
]);
